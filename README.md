\# Телефонная сеть



Клиент-серверное приложение для регистрации звонков и управления абонентами.



\## Описание



Система регистрирует звонки с домашних и мобильных номеров и автоматически списывает с баланса израсходованную сумму. В системе реализована авторизация с хешированием паролей, управление устройствами, различные тарифные планы, а также административная панель.



\## Технологии



\- Python 3.x

\- Tkinter (GUI клиент)

\- Flask (REST API сервер)

\- SQLite (база данных)

\- unittest (тестирование)



\## Функциональность



\### Для пользователя

\- Авторизация с хешированием паролей

\- Пополнение баланса

\- Управление устройствами (добавление/удаление)

\- Регистрация звонков

\- История звонков

\- Просмотр клиентов



\### Для администратора

\- Создание/удаление клиентов

\- Изменение тарифов клиентов

\- Установка баланса любому клиенту

\- Управление тарифами

\- Просмотр статистики
